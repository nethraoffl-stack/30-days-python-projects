# Day 12 - ATM Simulator

A Python program that simulates basic ATM operations through a command-line interface.

## Features

- View account balance
- Deposit money
- Withdraw money
- Prevents transactions when balance is insufficient
- Interactive menu-driven system
- Exit option

## Operations Available

- View Balance
- Deposit
- Transaction (Withdraw)
- Exit

## Built With

- Python
