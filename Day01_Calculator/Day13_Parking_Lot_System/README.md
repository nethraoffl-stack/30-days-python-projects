# Day 13 - Parking Lot Management System

A Python program that simulates a simple parking lot management system through a command-line interface.

## Features

- Park a vehicle
- Remove a vehicle
- View available parking spaces
- Prevent parking when the lot is full
- Check whether a vehicle is parked
- Exit option

## Operations Available

- Park a Vehicle
- Remove a Vehicle
- View Available Parking Spaces
- Exit

- Python
