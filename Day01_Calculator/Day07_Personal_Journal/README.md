# Day 7 - Personal Journal

A simple journal application that allows users to write and view journal entries stored in a text file.

## Features

* Write journal entries
* View all previous entries
* Persistent storage using a text file
* Menu-driven interface

## Concepts Used

* Functions
* File handling
* Reading and writing text files
* Loops and conditionals

## Built With

* Python
