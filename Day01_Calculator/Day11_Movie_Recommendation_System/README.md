# Day 11 - Movie Recommendation System 

A Python program that recommends movies based on the user's mood using random selection.

## Features

* 8 different moods supported
* Random movie recommendations
* Interactive user input
* Option to get recommendations multiple times
* Handles invalid mood entries

## Moods Available

* Happy
* Sad
* Bored
* Motivated
* Romantic
* Excited
* Stressed
* Adventurous

## Built With

* Python
* random module
